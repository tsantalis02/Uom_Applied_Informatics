Πληροφορίες εκτέλεσης

Το μόνο που χρειάζεται να κάνουμε είναι να ανοίξουμε το αρχείο 
του κώδικα σε Jupyter Notebook ,το οποίο είναι απαραίτητο να 
υποστηρίζει την γλώσσα Python. Εφόσον ανοίξουμε επιτυχώς το αρχείο 
στη συνέχεια απλώς εκτελούμε(Run) ένα-ένα τα κελιά(cells) τα οποία 
περιέχουν τον κώδικα.
